# 开始使用

TIP

CasaOS 完全支持 ZimaBoard、Intel NUC 迷你电脑和 Raspberry Pi。此外，更多的计算机和开发板，并通过单线安装与 Ubuntu、Debian、Raspberry Pi OS 和 CentOS 完全兼容。

# 快速设置

从上面的列表中全新安装系统并运行以下命令：

wget -qO- https://get.casaos.io | sudo bash
1

或

curl -fsSL https://get.casaos.io | sudo bash
1

# 卸载CasaOS

v0.3.3 或更高版本

casaos-uninstall
1

v0.3.3之前

curl -fsSL https://get.icewhale.io/casaos-uninstall.sh | sudo bash
1
