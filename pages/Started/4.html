# 玩转CasaOS-CasaOS实际使用场景

# 一. 搭配玩客云等ArmV7设备搭建个人云

准备工作：一台玩客云，一根网线，一条公对公USB连接线，一个镊子/铁丝，一个烧录镜像工具(这里我用的是USB_Burning_Tool)，Armbian镜像

# 第一步：下载安装烧录镜像工具

先在电脑上安装烧录镜像工具，下载Armbian镜像到能找到的文件夹中

# 第二步：玩客云连接电脑

将玩客云主板拆出来，将公对公USB连接线接入电脑，准备电源线和镊子/铁丝，一只手先将镊子/铁丝找到需要短接的触点（玩客云主板分为V1.0和V1.3，短接触点不同记得网上搜一下具体位置）进行短接操作，另一只手插入电源线，出现主板指示灯不亮同时电脑有USB设备接入的声音则连接成功，此时可以放开短接的那只手，发现USB_Burning_Tool提示连接成功即为成功连接电脑

# 第三步：导入镜像开始刷机

将img后缀的Armbian镜像导入USB_Burning_Tool，点击开始即可刷机，期间提示红色可以拔掉USB并重新短接连接，直至刷机成功

# 第四步：接入路由器初始化

将玩客云插上电并与路由器连接，在路由器后台找到玩客云的内网ip，使用远程SSH工具建立与玩客云的远程连接，根据提示输入用户名“root”和初始密码1234即可登录，接下来修改默认密码，修改完后完成一些基本设置就进入了玩客云系统，使用date-R检查时区，如果不对请使用cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime设置上海时区即可

# 第五步：安装并激活CasaOS

在ssh命令行输入wget -qO- https://get.casaos.io | bash即可安装CasaOS，如果没有科学上网环境会很慢，如果出现错误可以多装几遍，安装完成后会出现你的登录IP（其实就是玩客云的IP），在路由器输入即可进入CasaOS的Web设置界面，如图所示
