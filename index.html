# 欢迎来到 CasaOS Wiki。

  • CasaOS 是一款基于社区的开源软件，专注于提供简单的家庭云体验。
  • 如果您是新手，请随时开始安装 CasaOS。或者，您可以按照指南进行操作，以充分利用您的单板计算机、备用 PC 或任何运行 Linux 的设备
  • 如果您遇到任何问题，请随时尝试一些故障排除，或前往 Discord 或 GitHub 讨论板获得一些帮助。
  • 您还可以在我们的 Discord 中查看历史消息，这些消息为您可能遇到的任何问题积累了很多有价值的信息。
  • 如果您想提供帮助，请随时查看您可以参与的各个领域的贡献页面。
  • 最后但并非最不重要的一点是，如果您想结交朋友，请查看其他人的用户页面。😃

本文为非官方文档，仅为个人学习所用
